# Calculadora Python com Histórico
  Projeto desenvolvido para consolidar conhecimentos de **python 3**, com foco em modularização, tratamento de erros e manipulação de arquivos.

## Funcionalidades
- **Operações Matemáticas**: Soma, Subtração, Multiplicação e Divisão.
- **Histórico Persistente**: Salva automáticamente todas as contas realizadas em um arquivo '.txt' .
- **Interface Colorida**: Menu interativo utilizando códigos de cores ANSI no terminal.
- **Tratamento de Erros**: Sistema robusto que não encerra ao receber entradas inválidas ou erros de leitura de arquivo.

## Tecnologias Utilizadas
- **Python 3**
- **Bibliotecas**: 'os', 'time'
- **Conceitos**: pacotes (Mundo 3 - Curso em Vídeo), List Comprehension, Try/Except.
